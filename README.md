# Humor-Datasets

This repository provides official access links and acquisition instructions for the public datasets used in the paper:

**T-HumorAGSA: A Gated Anchor-Guided Self-Attention Model for Classroom Teacher Humor Language Detection**

> Note: This repository provides official links and acquisition instructions only. We do **not** redistribute third-party datasets.

---

## 1. ColBERT (Humor Dataset)

- Paper: https://doi.org/10.1016/j.eswa.2024.123685  
- Dataset access: Please refer to the original paper and associated repository provided by the authors.

## 2. SemEval 2021 Task 7 (Humor and Offense)
